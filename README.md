# ControlsCon2026
This will be a repository for Controls Con Stations given out. 

All stations that are put in this repository are not supported by Lynxspring. Use at your own risk.

The passphrases for all stations will be Admin12345

All stations contain graphics (using kitPx, kitPxN4Svg, and SVGs from Material Design Icons library - found at pictogrammers.com, Alarms, Histories, and Tagging (using the haystack4 tag dictionary).

These stations will get you roughly 80% there. There will be some modifications for alarms, and some tweaks to sequences of opertations. 

The VAV_AHU contains a Chilled Water VAV System, but could easily be converted to DX with relative ease.
